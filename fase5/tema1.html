Etiquetas Básicas de HTML

Según Celaya (2014) las etiquetas de HTML pueden ser distribuidas en:

  • Etiquetas Básicas

    <p> ... </p> Se utiliza para representar un párrafo
    <br> Salto de linea
    <Strong> ... </Strong> Hace referencia a algo muy importante
    <h1> ... </h1> Se utiliza para representar el encabezado de una página
    <ul> ... </ul> Lista de elementos donde el orden no tiene relevancia
    <ol> ... </ol> Lista de elementos donde el orden si importa
    <li> ... </li> Representa un elemento de la lista

  • Etiquetas Semánticas

    <header> ... </header> Para definir la cabecera de página
    <footer> ... </footer> Se utiliza para definir el pie de página
    <nav> ... </nav> Esta etiqueta permite definir un elemento de navegación de la página web
    <section> ... </section> Se utiliza para encerrar el código correspondiente a una sección genérica dentro de un documento
    <aside> ... </aside> Delimita todo aquel contenido que no es considerado como contenido principal de la página (bloques publicitarios, citas, calendarios, etc.)
    <article> ... </article> Define básicamente el contenido de la página web (noticias, entradas de un blog, etc.)

  • Etiquetas de Cointenido Multimedia

    <video> ... </video> Integra un vídeo en la página web
    <audio> ... </audio> Inserta un archivo mp3 en la página web
    <canvas> ... </canvas> Esta etiqueta siver como plataforma alternativa para gráficos y animaciones
