El proposito de este sitio web es la exposición de conceptos relacionados con Etiquetas Básicas de HTML, atributos y valores


Introducción

Las etiquetas son la estructura básica del HTML. Estas etiquetas son compuestas por atributos y el contenido.

HTML contempla mas de 100 etiquetas de las cuales algunas ya son obsoletas para HTML5. La gran mayoría de etiquetas van en parejas, iniciando la primera entre angulos y terminando del mismo modo solo que con una barra inclinada derecha. Cuando la etiqueta por si sola no contiene suficiente información se procede a utilizar los atributos que son pares nombre-valor separados por "=" y escritos en la etiqueta inicial de un elemento después del nombre de dicho elemento. El valor puede estar entre "comillas dobles" o 'simples'.
